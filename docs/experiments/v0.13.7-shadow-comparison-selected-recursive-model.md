# v0.13.7 repeat shadow-mode comparison with selected recursive model

## Purpose

v0.13.7 evaluates whether the v0.13.6 selected recursive model should become the candidate surrogate plant model for the future offline controller simulator.

This stage compares the placeholder baseline, the v0.13.4 one-step refit model, and the v0.13.6 selected recursive model under the same offline shadow-mode replay window.

## Methodological context

- v0.13.4 produced a one-step refit model.
- v0.13.5 compared placeholder and one-step refit in shadow mode and produced a caution result.
- v0.13.6 refit recursive plant-model candidates and selected a recursive model with a pass result.
- v0.13.7 repeats the shadow comparison with the selected recursive model included.

## Shadow-mode protocol

- Mode: recursive shadow rollout.
- Evaluation window: v0.13.6 validation segment.
- Initial recursive state: first observed value of the validation segment.
- Controller policy: proportional shadow recommendation only.
- Actuator application: false.

## Anti-leakage constraint

For prediction at target row `i`, the model uses only the previous recursive state and command history available before row `i`. The observed target value at row `i` is not used before producing the prediction.
