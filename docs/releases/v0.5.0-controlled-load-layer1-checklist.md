## Security checks

Before publication, confirm that the repository does not contain:

- [ ] `payer.json`;
- [ ] keypair JSON files;
- [ ] Kubernetes Secret material;
- [ ] kubeconfig files;
- [ ] `admin.conf`;
- [ ] `.env.controlled-load`;
- [ ] private SSH keys;
- [ ] accidental local smoke extraction files.

Recommended command:

```bash
grep -RInE 'BEGIN OPENSSH PRIVATE KEY|BEGIN RSA PRIVATE KEY|client-key-data|certificate-authority-data|private_key|secret' \
  . \
  --exclude-dir=.git || true
```

Also check tracked files:

```bash
git ls-files | grep -Ei 'payer|keypair|kubeconfig|admin.conf|secret|\.env' || true
```
